# Escapenote Web

## ✨ Requirements

- node 16.16.0
- npm 8.14.0

## 📚 Technical Stacks

- React
- Nextjs
- Typescript
- Redux Toolkit
- React Query
- Emotion
- Axios
- Jest

## 🌩 AWS Architecture

- CloudFront
- S3
- Lambda@Edge

## 🚡 CI / CD

- Github Actions

## 📦 Install

```shell
$ npm install
```

## 🔨 Runs

```shell
# development
$ npm run dev

# production mode
$ npm run build
$ npm run start
```

## ⌨️ Testing

```shell
# type check
$ npm run type-check

# lint
$ npm run lint

# test
$ npm run test

# test: watch mode
$ npm run test:watch
```
